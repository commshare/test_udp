# test_udp

测试比较 kcp 、 raknet、enet、tcp 等库

## test1

- 测试内容

  服务器以 `一定频率` 发送 400 字节的数据包 (如 66 毫秒)，查看客户端前后2个包的时间间隔

- 参考意义

  帧同步游戏，服务器会定时广播数据包给客户端，因此能考察服务器的每帧其他玩家指令包到达本机的时间。

- 测试内容
